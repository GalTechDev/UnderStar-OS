Plugin Structure

A plugin is a folder inside plugins/ containing an __init__.py with a class inheriting from Plugin.

from understar.core.plugin import Plugin

class MyPlugin(Plugin):
    def on_load(self):
        self.logger.info("Plugin Loaded!")

Slash Commands

Use the @slash_command decorator to register commands.

from understar.core.plugin import slash_command
import discord

@slash_command(name="ping", description="Replies with Pong")
async def ping(self, interaction: discord.Interaction):
    await interaction.response.send_message("Pong! üèì")

Event Listener

Listen to Discord events strict typing using the Event enum.

from understar.core.plugin import listen
from understar.core.events import Event

@listen(Event.ON_MESSAGE)
async def on_message(self, message):
    if message.author.bot: return
    print(message.content)

Scheduled Tasks

Run background tasks at specific intervals.

from understar.core.plugin import task

@task(minutes=5)
async def background_task(self):
    self.logger.info("Running task...")

Data Persistence

Save and load data easily without a database.

# Save
self.data.save({"coins": 10}, scope="user", id=123)

# Load
coins = self.data.get(scope="user", id=123, default={})
