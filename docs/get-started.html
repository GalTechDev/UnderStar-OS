Key Concepts

ğŸ§© Plugin

A Python class that inherits from Plugin. It has access to the logger, data manager, and can declare commands/events.

âš¡ Event Bus

Communication system between plugins. A plugin can emit an event, others can listen without direct dependency.

ğŸ’¾ Data Manager

Automatic JSON storage. No file management needed, the framework scopes data by guild, user, or global.

âš¡ Quick Start

Create a file named main.py:

from understar import OS

if __name__ == "__main__":
    bot = OS()
    bot.start()

ğŸ’¡ What does this code do?

Line 1: Imports the main class of the framework.
Line 4: Creates an instance of your bot's "operating system".
Line 5: Starts the main loop. On first launch, the token will be requested and saved automatically.

Run the bot:

python main.py
