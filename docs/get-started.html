Get Started

Learn how to install and start building with UnderStar-OS V2 in minutes.

Installation

UnderStar-OS is available on PyPI. You can install it using pip:

pip install understar

You also need git installed on your system for plugin management features.

Quick Start

Create a new file named main.py (or exemple.py) and add the following code: 

from understar import OS

if __name__ == "__main__":
    # Initialize the bot
    bot = OS()
    
    # Start the bot
    # The token will be requested on the first run
    # and saved in data/token/bot_token
    bot.start()

Run the script:

python main.py

Project Structure

Once you run the bot, it will create the necessary directories for you:

my-bot/
├── main.py             # Your entry script
├── data/               # Stores tokens, configs, and databases
└── plugins/            # Place your custom plugins here
    └── my_plugin/
        └── __init__.py

You are now ready to write your first plugin! Check out the Documentation.
